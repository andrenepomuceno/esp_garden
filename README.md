# ESP Garden

Automatic irrigation and monitoring system.
- Based on NodeMCU ESP32S
- Using PlatformIO/Arduino IDE.

![First Protype](docs/2021-05-01_15.02.36.jpeg)

## Features

- Local WebServer
- Cloud based logging
